# AI Caption Generator

A simple AI-powered web application that generates social media captions based on a given topic, platform, and tone.

## ✨ Features
- Generate AI captions using Groq API (free)
- Select platform (Instagram, Facebook, LinkedIn)
- Choose tone (Friendly, Professional, Casual)
- Clean modern UI based on Figma design
- Smooth animations and transitions
- Copy caption functionality
- Regenerate captions

## 🛠 Tech Stack
- Laravel (Backend)
- Vue 3 + Inertia (Frontend)
- Tailwind CSS (Styling)
- Groq API (AI)
